## Visão Geral

**SlimChoco** é uma landing page de alta conversão para um chocolate emagrecedor funcional, construída com Django 5.x full stack e Tailwind CSS.

- Backend: Python 3.12+ / Django 5.x
- Frontend: Django Template Language (DTL) + Tailwind CSS
- Banco de dados: SQLite (padrão Django)
- Interface: 100% em português brasileiro

O código-fonte Python é escrito em inglês. A interface com o usuário é em português brasileiro.
